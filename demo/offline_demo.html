แบบทดสอบความถูกต้องของคำตอบ AI

AI Answer Correctness Test / เอาคำถามและคำตอบจาก AI ตัวอื่นมาตรวจว่าเชื่อถือได้ไหม. แพ็กเกจนี้ไม่รวม core source, private memory, raw data หรือ internal logic.

มันทำอะไร / What this tool does

เอาคำตอบจาก AI ตัวอื่นมาตรวจ ว่าควรเชื่อถือ ปล่อย หยุด ปฏิเสธ หรือหาหลักฐานเพิ่ม / Check an answer from another AI and decide whether it should be trusted, released, stopped, refused, or reviewed further.

ไม่ใช่อะไร / What it is not

ไม่ใช่ช่องถาม AI ใหม่ ไม่ใช่เครื่องคิดเลข และไม่ใช่ ELUZA AI ตัวเต็ม / It is not a new chatbot, not a calculator, and not full ELUZA AI.

1. เปิด AI แล้วถาม ใช้ ChatGPT, Gemini, Claude หรือ AI อื่นถามคำถามจริงก่อน / Ask a real question in any AI.
2. ใส่คำถามช่อง 1 คัดลอกคำถามเดิมมาใส่ / Paste the original user question.
3. ใส่คำตอบช่อง 2 คัดลอกคำตอบร่างของ AI มาใส่ / Paste the AI draft answer.
4. ใส่กฎ/อ้างอิงช่อง 3 ใส่กฎ ข้อบังคับ หลักฐาน หรือแหล่งอ้างอิง แล้วกดตรวจ / Add rules, evidence, or references, then evaluate.
5. ดูผลและประเมิน ดูว่าคำตอบควรปล่อย หยุด หรือหาหลักฐานเพิ่ม แล้วตอบแบบประเมิน / Read the result and send feedback.
1. ผู้ใช้ถาม / User asks คำถามหรือคำขอจากผู้ใช้ / Prompt or request
2. AI ร่างคำตอบ / AI drafts คำตอบร่างจาก AI/LLM / Candidate answer
3. เครื่องมือตรวจ / Tool checks ชั่งหลักฐาน สิทธิ์ ความเสี่ยง / Evidence, permission, risk
4. คำตอบที่ปลอดภัยกว่า / Safer output ปล่อย / ปฏิเสธ / วิจัยต่อ / Allow, refuse, research

เลือกสถานการณ์ทดสอบ / Choose a test case

สำหรับคนทั่วไป: เริ่มตรงนี้ก่อน กดตัวอย่างหนึ่งอัน แล้วดูว่าระบบควรปล่อยคำตอบ AI หรือควรหยุดก่อน / For general testers: start here, choose a case, and see whether the AI answer should be released or stopped.

บททดสอบสำหรับคนทั่วไป / General Test Script

  1. กดการ์ดตัวอย่างหนึ่งอัน เช่น `ต้นไม้ / น้ำบาดาล` หรือ `ขอ source code`
  2. ดูช่อง `ก่อนตรวจ` ว่า AI ร่างคำตอบไว้แบบไหน
  3. ดูช่อง `หลังตรวจ` ว่าเครื่องมือแก้ให้ปลอดภัยขึ้นหรือหยุดคำตอบหรือไม่
  4. อ่าน `ความหมายแบบคนทั่วไป` ว่าเหตุผลเข้าใจง่ายหรือไม่
  5. ตอบแบบประเมินท้ายหน้า: เข้าใจไหม, งงตรงไหน, มีประโยชน์ไหม

ตรวจคำตอบ AI / Check an AI answer

ถาม AI ตัวไหนก็ได้ก่อน แล้วเอาคำถาม คำตอบ และหลักเกณฑ์ที่อยากให้ยึดมาใส่ 3 ช่องนี้ / Ask any AI first, then paste the question, the AI answer, and the rule or reference to check.

ไม่ใช่ช่องถาม AI ใหม่ แต่เป็นช่องตรวจคำตอบที่ AI ตอบมาแล้ว / This does not generate a new answer. It checks an answer that an AI already gave.
เรื่องสำคัญ เช่น กฎหมาย แพทย์ การเงิน ศาสนา หรือความปลอดภัยจริง ยังต้องให้ผู้เชี่ยวชาญตรวจ / Important domains still require qualified review.
