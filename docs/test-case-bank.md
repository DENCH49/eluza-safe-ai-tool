# Test Case Bank / ธนาคารโจทย์ทดสอบ

หน้านี้ช่วยคนที่ “ไม่รู้จะถามอะไร” ให้เริ่มทดสอบ ELUZA Safe AI Tool ได้ทันที

วิธีใช้:

1. เลือกเคสด้านล่าง
2. เอาช่อง 1 ไปถาม AI ตัวใดก็ได้ก่อน
3. เอาคำตอบ AI ที่ได้จริงมาใส่ช่อง 2
4. เอาช่อง 3 ไปใส่เป็นหลักเกณฑ์
5. กด `ตรวจคำตอบ / Check`
6. ส่งผลเข้า Google Form ถ้าต้องการช่วยทดสอบ

ถ้ายังไม่มี AI ตัวอื่นให้ถาม สามารถใช้คำตอบตัวอย่างในช่อง 2 ได้
